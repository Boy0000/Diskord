pngDefaultAvatar

fun User.pngDefaultAvatar(size: Int? = null): String

Gets the url of the user's default avatar in PNG format.

The default avatar may not be used if the user has uploaded a custom avatar.

Parameters

size - the size of the image. Must be a power of two or null.

Exceptions

DiscordBadRequestException - when the size is not null or a power of two.

Return
the url of the png formatted avatar.
