fun Guild.jpgIcon(size: Int? = null): String?

Gets the url of the guild icon in JPG format. Null if there is no custom icon.

Parameters

size - the size of the image. Must be a power of two or null.

Exceptions

DiscordBadRequestException - when the size is not null or a power of two.

Return
the url of the jpg formatted emoji or null if there is no icon.
