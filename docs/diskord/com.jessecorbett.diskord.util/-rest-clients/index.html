RestClients

abstract class RestClients<T : RestClient>

Generic container for RestClient instances.

Automatically creates and stores RestClient instances based on the id relevant to each client instance.

Parameters

T - the RestClient implementation class.

userToken - the user token used for authentication by each RestClient.

gen - a lambda which returns a new instance of class T when requested by the user.
