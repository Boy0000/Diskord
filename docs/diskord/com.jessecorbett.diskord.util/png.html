fun Emoji.png(size: Int? = null): String?

Gets the url of the emoji in PNG format, null if the emoji is a unicode character.

Parameters

size - the size of the image. Must be a power of two or null.

Exceptions

DiscordBadRequestException - when the size is not null or a power of two.

Return
the url of the png formatted emoji or null if it is unicode.
